LigaçõesQuímicas

Elas são a união dos átomos para formar substâncias químicas. Ocorrem quando os átomos dos elementos químicos combinam entre si, os principais tipos de ligações químicas são:

• Ligações iônicas: É a transferência de elétrons, ocorre quando um átomo doa elétrons para outro átomo.
• Ligações covalentes: É o compartilhamento de elétrons, ocorre quando átomos compartilham pares de elétrons.
• Ligações metálicas: É a presença de elétrons livres, eles mantêm íons metálicos unidos.
